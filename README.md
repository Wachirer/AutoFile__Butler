# AutoFile-Butler

Python automation script to organize files by type. Supports CLI options, config file, and dry-run mode.

## Features
- Sorts files into folders by type (Images, Documents, Videos, Code, Others)
- Configurable folders and extensions via `config.json`
- Dry-run mode to preview changes
- Logging of all actions
- CLI options for custom folder paths

## Usage
### Dry-run (preview)

python main.py --dry-run
